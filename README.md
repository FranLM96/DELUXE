# Deluxe | Elite Music Experience

Plataforma de venta de boletos para eventos de música electrónica. Built with React, TypeScript, Tailwind CSS y Vite.

**Live:** https://franlm96.github.io/DELUXE/

## Features

✨ Responsive design optimizado para móviles y desktop
✨ Menú desplegable elegante con transiciones suaves  
✨ Venta de boletos con selector de cantidad
✨ Detalles completos de eventos con lineups
✨ Soporte multiidioma (Español/Inglés)
✨ Integración con Google Maps
✨ Compartir eventos en redes sociales

## Run Locally

**Requisitos:** Node.js (v18+)

```bash
# Clonar repositorio
git clone https://github.com/FranLM96/DELUXE.git
cd DELUXE

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Build para producción
npm run build

# Preview de la build
npm run preview
```

## Project Structure

```
src/
├── components/     # Componentes reutilizables
├── context/        # React Context (Auth, Language)
├── hooks/          # Custom hooks
├── i18n/           # Traducciones
├── pages/          # Páginas de la aplicación
├── data/           # Mock data
└── types.ts        # TypeScript interfaces
```

## Tech Stack

- **Frontend:** React 19, TypeScript
- **Styling:** Tailwind CSS 4 (Production-ready)
- **Build:** Vite 6
- **Routing:** React Router v7
- **Icons:** React Icons
- **Deploy:** GitHub Pages
